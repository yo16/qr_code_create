## 看板にQRコードを掲載するメリット

看板や屋外広告は、通行人の目に入りやすい強力な広告媒体です。しかし、看板だけでは詳細情報を伝えるのに限界があります。QRコードを掲載すれば、通行人がスマートフォンでスキャンするだけで、Webサイトやキャンペーンページにアクセスできます。

UTMパラメータを設定すれば、看板からのアクセスを正確に計測でき、看板広告の費用対効果を数値で把握できます。

## 看板用QRコードの推奨UTM設定

| パラメータ | 推奨値 | 説明 |
|---|---|---|
| utm_source | `signage` | 看板からのアクセスを識別 |
| utm_medium | `qr` | QRコード経由であることを識別 |
| utm_campaign | `store_signage_2026` | 看板施策の種類で管理 |

### 設置場所ごとの計測

複数の場所に看板を設置する場合、`utm_content` で場所を区別しましょう。

| 設置場所 | utm_content |
|---|---|
| 店頭看板 | `storefront` |
| 駐車場 | `parking` |
| 道路沿い | `roadside` |
| ビル壁面 | `building` |

## 看板用QRコードのおすすめデザイン

### プリセット: ポップ

看板は遠くからでも目立つことが大切です。QR Code Createの「ポップ」プリセットは、鮮やかな色合いで視認性が高く、屋外の看板に適しています。

### デザインのコツ

- **大きく、シンプルに**: 看板のQRコードは遠距離からスキャンされるため、できるだけ大きく、ドットがはっきりしたデザインにしましょう
- **高コントラスト**: 太陽光の下でもスキャンできるよう、前景色と背景色のコントラストを最大にしてください。黒白が最も確実です
- **キャプション**: 「スマホでスキャン」「QRコードを読み取って詳細を見る」など、大きなフォントで行動を促しましょう
- **フレームは控えめに**: 遠距離では装飾的なフレームが判読しにくくなることがあります。シンプルなフレームか、フレームなしがおすすめです

## 看板への印刷サイズの目安

看板は読み取り距離が長いため、QRコードを大きく印刷する必要があります。

| 看板サイズ | QRコード推奨サイズ | 読み取り距離 |
|---|---|---|
| 店頭スタンド看板 | 50mm x 50mm以上 | 30cm〜1m |
| 店頭壁面看板 | 100mm x 100mm以上 | 1〜2m |
| 道路沿い看板 | 150mm x 150mm以上 | 2〜3m |
| 大型屋外広告 | 200mm x 200mm以上 | 3m以上 |

### 読み取り距離とQRコードサイズの関係

一般的な目安として、**読み取り距離の1/10以上**のサイズでQRコードを印刷すると安定してスキャンできます。例えば、2m離れた場所からスキャンしてもらう場合は、200mm（20cm）以上のサイズが推奨です。

### ダウンロード形式の選び方

大型の看板制作には、**SVG形式**が必須です。ベクター形式なので、どんなサイズに拡大しても画質が劣化しません。看板製作会社への入稿には**PDF形式**も対応しています。

## 看板用QRコードの活用例

- **店舗のWebサイト**: 営業時間、メニュー、サービス内容の確認
- **Googleマップ**: 店舗の場所と経路案内を表示
- **期間限定キャンペーン**: セールや特別イベントの告知ページにリンク
- **求人情報**: 店頭看板に求人ページへのQRコードを掲載
- **テイクアウト注文**: 外からテイクアウトメニューを確認・注文
- **SNSフォロー**: Instagram等のSNSアカウントに誘導

## 屋外看板QRコードの注意点

1. **耐候性**: 屋外の看板はQRコードが日焼けや雨で劣化することがあります。定期的に読み取りテストを行いましょう
2. **反射防止**: ガラスやアクリル越しの場合、光の反射でスキャンしにくくなることがあります。反射防止素材の使用を検討してください
3. **夜間の視認性**: 照明がない場所では、スマートフォンのライトで照らしてもスキャンできるよう、十分なサイズと高コントラストを確保しましょう
4. **高さの配慮**: 看板の設置高さは、スマートフォンを構えやすい位置（地面から1〜1.5m程度）にQRコードを配置すると理想的です

## まとめ

看板にQRコードを掲載すると、オフラインの広告効果をオンラインの行動につなげられます。十分なサイズと高コントラストで印刷し、設置場所ごとにUTMパラメータを分ければ、看板広告のROIを正確に計測できます。

QR Code Createでは、UTMパラメータの参照元で「店内POP」を選び、装飾で「ポップ」プリセットを適用すれば、看板に適した目立つQRコードを作成できます。
