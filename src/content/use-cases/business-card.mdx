## なぜ名刺にQRコードを入れるのか

名刺交換は対面でのビジネスコミュニケーションの基本ですが、紙の名刺だけでは自社のWebサイトやサービスへの誘導が難しいのが課題です。QRコード入り名刺を使えば、相手がスマートフォンでスキャンするだけでWebサイトにアクセスでき、URLを手入力する手間がなくなります。

さらに、UTMパラメータ付きのQRコードを使えば、名刺からどれだけのアクセスがあったかをGoogle Analyticsで正確に計測できます。

## 名刺用QRコードの推奨UTM設定

名刺用のQRコードには、以下のUTMパラメータを設定することをおすすめします。

| パラメータ | 推奨値 | 説明 |
|---|---|---|
| utm_source | `business_card` | 名刺からのアクセスであることを識別 |
| utm_medium | `qr` | QRコード経由であることを識別 |
| utm_campaign | `networking_2026` | 年度を含めると管理しやすい |

QR Code Createでは、UTMパラメータの参照元で「名刺」を選ぶだけで、上記の設定が自動的に入力されます。

## 名刺用QRコードのおすすめデザイン

名刺はビジネスの場で使うものなので、落ち着いたデザインがおすすめです。

### プリセット: ビジネス

QR Code Createの「ビジネス」プリセットは、紺色ベースのシンプルなデザインです。名刺のフォーマルな雰囲気に馴染みます。「詳しくはこちら」のキャプションも自動で付きます。

### 名刺デザインのポイント

- **色は落ち着いたトーン**: 名刺の背景色に合わせて、QRコードの前景色を調整するとデザインに統一感が出ます
- **ロゴの追加**: 会社のロゴをQRコードの中央に入れると、ブランド認知を高められます。ロゴサイズは15%程度がおすすめです
- **フレーム**: ビジネスフレームを選ぶと、枠線付きで名刺に配置しやすくなります

## 名刺への印刷サイズの目安

名刺は小さな媒体のため、QRコードのサイズ選びが重要です。

| 項目 | 推奨値 |
|---|---|
| 最小サイズ | 15mm x 15mm |
| 推奨サイズ | 20mm x 20mm |
| 周囲の余白 | 最低2mm |
| 推奨解像度 | 2x以上 |

QRコードが小さすぎるとスキャンできない場合があります。15mm以下にならないよう注意してください。

### ダウンロード形式の選び方

名刺のデザインデータに組み込む場合は、**SVG形式**がおすすめです。拡大・縮小しても画質が劣化しないため、デザインソフトでの配置が容易です。印刷用のPNG形式を使う場合は、**2x以上の解像度**を選んでください。

## 名刺用QRコードの活用例

- **自社Webサイトへの誘導**: 会社のトップページや自己紹介ページへリンク
- **ポートフォリオへの誘導**: デザイナーやエンジニアが作品集へリンク
- **SNSプロフィールへの誘導**: LinkedInやXのプロフィールページへリンク
- **vCard（電子名刺）の配布**: 連絡先情報を直接スマートフォンに保存

## まとめ

名刺にQRコードを印刷することで、紙からデジタルへのスムーズな導線を作れます。UTMパラメータを設定すれば、名刺経由のアクセスを計測でき、営業活動の効果測定にも役立ちます。

QR Code Createでは、UTMパラメータの参照元で「名刺」を選び、装飾で「ビジネス」プリセットを適用すれば、最適な設定とデザインで名刺用QRコードを作成できます。
