## レシートにQRコードを活用するメリット

レシートはお客様が必ず受け取る紙媒体です。ここにQRコードを印刷すれば、会計直後のタイミングでWebサイトやクーポンページへ誘導できます。購入直後はお客様の関心が高い時間帯なので、アンケートやSNSフォローへの誘導も効果的です。

UTMパラメータを設定すれば、レシート経由のアクセス数を正確に計測できます。クーポンの利用率やアンケートの回答率を把握し、施策の改善に活かしましょう。

## レシート用QRコードの推奨UTM設定

| パラメータ | 推奨値 | 説明 |
|---|---|---|
| utm_source | `receipt` | レシートからのアクセスを識別 |
| utm_medium | `qr` | QRコード経由であることを識別 |
| utm_campaign | `receipt_coupon_2026` | レシート施策の種類で管理 |

## レシート用QRコードのおすすめデザイン

### プリセット: ビジネス

レシートは白黒印刷が多いため、シンプルで読み取りやすいデザインが重要です。QR Code Createの「ビジネス」プリセットをベースに、色を黒白に調整するのがおすすめです。

### デザインのコツ

- **白黒を基本に**: レシートプリンターは白黒が主流です。前景色は黒、背景色は白のままにしましょう
- **ロゴは控えめに**: レシートのQRコードは小さいため、ロゴを入れる場合は10%程度に抑えましょう
- **キャプション**: 「クーポンはこちら」「アンケートにご協力ください」など、スキャンの動機を明確にするテキストが効果的です

## レシートへの印刷サイズの目安

レシートの幅は一般的に58mmまたは80mmです。QRコードのサイズはレシートの幅に合わせて設定しましょう。

| レシート幅 | QRコード推奨サイズ | 注意点 |
|---|---|---|
| 58mm（標準） | 20mm x 20mm | 左右に余白を確保 |
| 80mm（ワイド） | 25mm x 25mm | 余裕をもって配置可能 |

### 印刷品質の注意

レシートプリンターは解像度が低い場合があります。URLが長いとQRコードのドットが細かくなり、読み取りにくくなることがあります。

- **URLを短くする**: 不要なパラメータを省略するか、短縮URLを使用
- **誤り訂正レベル**: QR Code Createではレベル「H」（高）で生成されるため、多少の印刷劣化にも対応できます

### ダウンロード形式の選び方

レシートプリンターへの組み込みには、**PNG形式（2x解像度）**が一般的です。POSシステムの仕様に合わせて形式を選んでください。

## レシート用QRコードの活用例

- **次回来店クーポン**: 割引クーポンのページに誘導してリピート来店を促進
- **満足度アンケート**: Googleフォーム等のアンケートに誘導して顧客の声を収集
- **LINE公式アカウント**: 友だち追加ページに誘導してリピーターを囲い込み
- **ECサイト**: 店舗で購入した商品のオンラインストアに誘導
- **ポイントカード**: デジタルポイントカードの登録ページに誘導
- **レビュー投稿**: Googleマップやグルメサイトのレビューページに誘導

## まとめ

レシートにQRコードを印刷すると、購入直後のお客様にアプローチでき、リピート来店やフィードバック収集に効果的です。施策ごとにUTMパラメータを変えて計測すれば、最も反応の良い施策を特定できます。

QR Code Createでは、UTMパラメータを設定し、装飾で「ビジネス」プリセットを適用すれば、レシートに適したシンプルなQRコードを作成できます。
