## ダウンロード形式の種類

QR Code Createでは、3種類のファイル形式でQRコードをダウンロードできます。

- **PNG**: 画像ファイル形式
- **SVG**: ベクター形式
- **PDF**: 印刷向けドキュメント形式

それぞれの特徴と適した用途を紹介します。

## PNG形式

PNGは、もっとも一般的な画像ファイル形式です。Webサイトへの掲載やSNSでの共有、デジタル媒体での使用に適しています。

### PNGが向いている用途

- Webサイトやブログへの掲載
- メールへの添付
- SNSでの共有
- プレゼンテーション資料への挿入

### 注意点

PNGはラスター形式（ドットの集まり）のため、大きく拡大すると画質が粗くなります。印刷物に使う場合は、解像度を高めに設定するか、SVGやPDF形式の利用を検討してください。

## SVG形式

SVGは、ベクター形式の画像ファイルです。拡大・縮小しても画質が劣化しないのが最大の特徴です。

### SVGが向いている用途

- 大きく印刷するポスターや看板
- サイズが未定の段階でのデータ作成
- デザインソフト（Illustrator、Figmaなど）での編集
- Webサイトでの高品質表示

### 注意点

SVGファイルを直接開けないアプリケーションもあります。メールに添付する場合や、対応していないソフトで使う場合はPNG形式を選んでください。

## PDF形式

PDFは、印刷物への入稿に適したドキュメント形式です。印刷会社への入稿や、印刷物のレイアウトに直接貼り付ける場合に便利です。

### PDFが向いている用途

- チラシやパンフレットへの入稿
- 印刷会社へのデータ納品
- 印刷物のレイアウトソフトへの配置
- 高品質な印刷出力

### 注意点

PDFファイルはWebサイトへの直接掲載には向いていません。Web用途にはPNGまたはSVGを使用してください。

## 解像度の選び方

PNG形式を選んだ場合、出力解像度を選択できます。3段階の解像度が用意されています。

### 通常（1x）

- 標準的な解像度です
- ファイルサイズ: 約20KB
- Webサイトやデジタル媒体での使用に適しています
- 画面上での表示には十分な品質です

### 2倍（2x）

- 通常の2倍の解像度です
- ファイルサイズ: 約80KB
- チラシや名刺など、一般的な印刷物に適しています
- Retinaディスプレイでの表示にも対応できます

### 4倍（4x）

- 通常の4倍の高解像度です
- ファイルサイズ: 約300KB
- ポスターや看板など、大きく印刷する場合に適しています
- 最高品質が必要な印刷物におすすめです

## 用途別おすすめ形式

| 用途 | おすすめ形式 | 解像度 |
|---|---|---|
| Webサイトに掲載 | PNG | 通常（1x） |
| メールに添付 | PNG | 通常（1x） |
| 名刺に印刷 | PNG または SVG | 2x以上 |
| チラシに印刷 | SVG または PDF | - |
| ポスターに印刷 | SVG または PDF | - |
| 印刷会社に入稿 | PDF | - |
| デザインソフトで編集 | SVG | - |

## まとめ

ダウンロード形式は用途に合わせて選びましょう。

- **Web・デジタル用途**: PNG形式（通常解像度）
- **一般的な印刷物**: PNG形式（2x以上）またはSVG形式
- **大判印刷・入稿**: SVGまたはPDF形式

迷った場合は、PNG形式の2x解像度を選んでおけば、多くの用途に対応できます。
