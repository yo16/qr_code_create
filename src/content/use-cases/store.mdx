## 店舗でQRコードを活用するメリット

店舗ビジネスでは、来店客にWebサイトやSNSへのアクセスを促す場面が多くあります。店内POP、レジ周り、テーブルなどにQRコードを設置すれば、お客様がその場でスマートフォンをかざすだけで情報にアクセスできます。

UTMパラメータを設定すれば、店内のどの場所に設置したQRコードが最もスキャンされているかを計測できます。効果の高い設置場所を特定して、店内のQRコード配置を最適化しましょう。

## 店舗用QRコードの推奨UTM設定

| パラメータ | 推奨値 | 説明 |
|---|---|---|
| utm_source | `store` | 店舗内からのアクセスであることを識別 |
| utm_medium | `qr` | QRコード経由であることを識別 |
| utm_campaign | `store_promo_2026` | 店舗プロモーション名で管理 |

### 設置場所ごとの計測

| 設置場所 | utm_content | 活用例 |
|---|---|---|
| レジ周り | `register` | LINE公式アカウント登録、ポイントカード |
| 店頭ウィンドウ | `window` | キャンペーン情報、メニュー表示 |
| 店内POP | `pop` | 商品詳細、レビューページ |
| テーブル | `table` | メニュー、注文ページ |
| レシート | `receipt` | クーポン、アンケート |

## 店舗用QRコードのおすすめデザイン

### プリセット: ビジネス

店舗のブランドイメージを損なわない落ち着いたデザインが適しています。QR Code Createの「ビジネス」プリセットは、紺色ベースのフォーマルなデザインで、どんな店舗にも馴染みます。

### デザインのコツ

- **店舗のブランドカラー**: QRコードの色を店舗のテーマカラーに合わせると統一感が出ます
- **店舗ロゴの追加**: QRコード中央にロゴを入れると、ブランド認知を高められます
- **説明テキスト**: QRコードの近くに「LINE友だち追加」「メニューを見る」など、スキャン先の内容を明記しましょう

## 店舗での印刷サイズの目安

| 設置場所 | QRコード推奨サイズ | 素材 |
|---|---|---|
| レジ横スタンド | 30mm x 30mm以上 | アクリルスタンド、POP |
| 店頭ウィンドウ | 50mm x 50mm以上 | ステッカー、ポスター |
| テーブルPOP | 25mm x 25mm以上 | ラミネートカード |
| 店内ポスター | 40mm x 40mm以上 | ポスター |

### ダウンロード形式の選び方

店内POPやスタンドに使う場合は、**PNG形式（2x解像度）**が扱いやすいです。大きめのポスターや看板には**SVG形式**を使うと、サイズ調整が自由にできます。

## 店舗用QRコードの活用例

- **LINE公式アカウントの友だち追加**: レジ横に設置してフォロワー獲得
- **Googleマップの口コミ投稿**: 口コミを書いてもらうページに誘導
- **デジタルメニューの表示**: 紙メニューの代わりにWebメニューを表示
- **クーポン配布**: 会計時にレシートと一緒にクーポンページへ誘導
- **会員登録・ポイントカード**: デジタル会員証の登録ページにリンク
- **Wi-Fi接続情報**: 店舗Wi-Fiのログインページへ誘導

## まとめ

店舗にQRコードを設置すると、お客様とのデジタル接点を手軽に増やせます。設置場所ごとにUTMパラメータを分けて計測すれば、最も効果的な場所を特定できます。

QR Code Createでは、UTMパラメータの参照元で設定し、装飾で「ビジネス」プリセットを適用すれば、店舗に合ったQRコードを作成できます。
